# Projeto em Html de um Trabalho Acadêmico:

Este é um site desenvolvido para apresentar um trabalho acadêmico sobre a sobrecarga feminina e o trabalho não remunerado. O conteúdo aborda:

- Introdução 
- Desenvolvimento 
- Conclusão
- Referências

## Tecnologias

- HTML5
- CSS3 (com fontes do Google Fonts - Roboto)
- Layout responsivo para diferentes tamanhos de tela

## Estrutura do site

- `header` → Título do trabalho
- `nav.menu` → Menu de navegação para seções do trabalho
- `main` → Conteúdo dividido em seções (`introducao`, `cap1`, `cap2`, `conclusao`, `referencia`)
- `section.ref-item` → Referências bibliográficas

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Navegue entre as seções pelo menu no topo.
3. As imagens e referências são ilustrativas e informativas para o trabalho acadêmico.

## Observações

- Site responsivo e otimizado para leitura.
- Design limpo e fontes legíveis, com destaque para títulos e seções.
